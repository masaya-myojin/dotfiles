# mmyoji's dotfiles

## Content

* .emacs.d
* .zshrc

## Install

```sh
# move your home path
$ cd

$ mkdir ./projects

$ cd ./projects

$ git clone git@github.com:masaya-myojin/dotfiles.git

$ cd

# create symbolic links
$ ln -s ./projects/dotfiles/.emacs.d ~/
$ ln -s ./projects/dotfiles/.zshrc ~/
```
